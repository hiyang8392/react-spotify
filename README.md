### 說明
仿 Spotify 音樂播放介面，音樂來源是使用官方 Youtube 連結，功能為播放音樂、搜尋專輯歌曲、新增歌曲至收藏庫、調整聲音大小與播放上/下一首歌曲。

### 技術
* React
* React-Player
* React-Redux
* Redux-Toolkit
* Styled-Components

### how to start
* `npm install`
* `npm start`
